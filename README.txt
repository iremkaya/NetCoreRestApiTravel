Post methodu yardımıyla güzergah ekleyebilirsiniz. 
Eğer güzergahı yayımlamazsanız o zaman get methodunda güzergah listesinde eklemiş olduğunuz güzergahı göremezsiniz.
Put methodu verilen id ye ait güzergahı yayımlama işlemi yapmaktadır.
GetAll methodu ile tüm listeyi çekebilirsiniz.
Delete methodunda girlen id değerini listeden silmek yerine pasife alarak silme işlemi yapılmıştır.
Join methodu ise bu güzergaha katılmak isteyen kişiler için geliştirilmiştir.


Not: 
Uygulama Swagger üzerinde çalışabilir durumdadır.
